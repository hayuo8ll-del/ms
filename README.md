# なつやすみ スタディ 🌻

小学**1年生**と**5年生**が、夏休み中に楽しく勉強を続けられる学習アプリ（PWA プロトタイプ）です。
インストール不要でブラウザで動き、ホーム画面に追加すればアプリのように使えて**オフラインでも動作**します。

## 特長

- **2学年モード**：1年生／5年生を切り替えて使用（学習データは学年ごとに別管理）
- **全4教科・約100問＋自動生成**：算数・国語・英語・理科社会（クイズ）
- **ゲーム感覚で継続**：正解でコイン獲得、連続学習日数（ストリーク）、11種類のバッジ、全問正解ボーナス、正解時の効果音＆紙吹雪演出
- **⏱️ タイムアタック**：制限時間（30/60/90/120秒）内に何問解けるか挑戦。**コイン2倍＋コンボボーナス**、自己最高記録を保存。教科ごと or 全教科ミックス。※学習統計・にがてプールには影響しません
- **🛍️ 着せ替えショップ**：ためたコインで**アバター（12種）**と**カラーテーマ（6種）**を交換。テーマはアプリ全体の配色が変わります
- **✍️ かきとり（手書き）練習＋📝 書き順アニメ**：キャンバスに指／マウス／スタイラスで書けるなぞり書きモード。**「▶ 書き順」で正しい筆順を1画ずつアニメ表示**（KanjiVG データ）。お手本の濃さを「こい／うすい／なし」で切替、田字格ガイド・画数表示付き。ひらがな・カタカナ・1年漢字（80字）／5年漢字。書けたらコイン獲得
- **🩹 にがて復習**：間違えた問題は自動でストックし、専用の復習モードで再挑戦。正解すると克服リストから消えます
- **🔊 音声よみあげ**：問題文を日本語で読み上げ（Web Speech API）。1年生はデフォルトON、各問題の🔊ボタンで聞き直しも可能
- **算数は自動生成**：毎回ちがう問題（1年＝たし算/ひき算/時計/数の大小、5年＝分数/小数/割合/平均/公倍数公約数/面積）
- **時計・図形は SVG で描画**：アナログ時計の読み取り、長方形の面積など
- **保護者メニュー**：教科別の正答率、直近14日の学習カレンダー、最近の記録、にがて問題数、問題数・効果音・よみあげの設定、データリセット
- **子ども向け UI**：大きなボタン、ふりがな、テンキー入力／4択
- **PWA・オフライン対応**：Service Worker でキャッシュ。PNG＋SVG のアイコン一式（`any`／`maskable`）を同梱

## 使い方（ローカル実行）

Service Worker を使うため `file://` ではなく HTTP サーバー経由で開きます。

```bash
# リポジトリのルートで
python3 -m http.server 8000
# ブラウザで http://localhost:8000/ を開く
```

スマホ／タブレットのブラウザで開き、「ホーム画面に追加」するとアプリとして使えます。

## 仕様（学習内容）

| 教科 | 1年生 | 5年生 |
|------|-------|-------|
| 算数 | たし算・ひき算（繰り上がり/繰り下がり）、10の合成、数の大小、時計（ちょうど/30分） | 分数の加算・約分、小数のかけ算/わり算、割合(%)、平均、最小公倍数・最大公約数、長方形/三角形の面積 |
| 国語 | ひらがな・カタカナ、1年配当漢字の読み、しりとり | 漢字の読み、四字熟語、慣用句、対義語 |
| 英語 | あいさつ・数・色・身近な単語 | 単語・曜日・意味・疑問文 |
| 理科社会 | 生活・自然のクイズ | 都道府県・県庁所在地、理科（光合成・蒸発など）、社会 |

問題は1回 5〜20問（保護者メニューで変更可）。算数は自動生成、その他は `js/data.js` の問題データから出題します。

## ファイル構成

```
index.html              画面の土台
css/style.css           スタイル
js/data.js              問題データ・バッジ・教科定義・ショップ・かきとり文字セット
js/generators.js        算数の問題自動生成（時計/図形の SVG 含む）
js/strokes.js           書き順ストロークデータ（KanjiVG 抽出, CC BY-SA 3.0）
js/app.js               アプリ本体（画面遷移・採点・スコア・ショップ・復習・よみあげ・かきとり・書き順・保存）
manifest.webmanifest    PWA マニフェスト
sw.js                   Service Worker（オフライン対応）
icons/                  アイコン（icon.svg と PNG 一式：96/192/512/512-maskable/apple-touch）
```

## データの保存

学習データ（コイン・ストリーク・バッジ・統計・ログ・設定）はブラウザの **localStorage**（キー `natsuyasumi_v1`）に保存されます。サーバー不要で、端末内に閉じています。
